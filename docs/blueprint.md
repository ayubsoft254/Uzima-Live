# **App Name**: Uzima Live

## Core Features:

- Multimodal User Interface: Provide a real-time user interface for live camera input (Vision) to scan documents like prescriptions or health cards, and microphone input (Voice) to capture user concerns.
- AI Document & Speech Interpretation Tool: Utilize Gemini Live API and Vision capabilities to interpret data from physical health records (e.g., prescriptions, health cards) and analyze spoken user queries for meaning.
- AI Personalized Guidance Generation Tool: Generate actionable, personalized health advice by synthesizing interpreted data from multimodal input with information fetched from the Uzima Mesh backend.
- Seamless Multilingual Output: Deliver AI-generated explanations and guidance audibly, switching dynamically and smoothly between English and Swahili to cater to user preferences.
- Uzima Mesh Backend Integration: Integrate with the existing Django-based Uzima 'Mesh' backend through Function Calling to retrieve community health trends and historical patient data for informed guidance.

## Style Guidelines:

- Primary color: A clear and trustworthy blue (#4DBBE6) to evoke professionalism and calm in a health context, while remaining vibrant.
- Background color: A very light, desaturated blue (#ECF4F7) for a clean, open, and unobtrusive canvas, ensuring maximum readability.
- Accent color: A fresh green-cyan (#10B389), analogous to the primary, to highlight interactive elements and convey vitality and positive affirmation.
- Body and headline font: 'Inter' (sans-serif) for its modern, objective clarity and excellent readability across varying screen sizes and text densities.
- Employ a set of clear, line-based icons that are intuitively understandable for health, communication, and real-time interaction functions, culturally sensitive and minimal in design.
- Design a clean, mobile-first interface optimized for real-time camera and microphone interaction, ensuring core functionality is always accessible and visible without clutter.
- Incorporate subtle, fluid animations for state changes and transitions, particularly during multimodal input detection and AI response delivery, to enhance the sense of real-time responsiveness.
